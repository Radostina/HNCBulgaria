HC-A2C Series PLC

HC-A2C, is high performance modular PLC. The extensible compact PLC is designed to seamlessly control communication network and analog positioning system.

HC-A2C, is high performance modular PLC. The extensible compact PLC is designed to seamlessly control communication network and analog positioning system. Maximum number of input/output points (including I/O points of extension blocks and the main unit) is 128 points, up to max. 384 points when HC-LINK is used, the HC-A2C uses its power and flexibility to provide solutions for variety of applications. 

1.3rd generation compact PLC 
2.Modular PLC, input/output extension blocks can be connected, up to 7 HCA2C Series special function blocks can be connected.
3.Support all-serises HC8C right expansion
3.High efficiency with more speed, more performance, more memory, and new functions. 
4.Built-in high speed processing and positioning 
5.Exchangeable communication expansion boards that mount directly into the main unit (RS-232C) 
6.Simultaneous memory cassette and display module mounting available 
7.Integrated real-time clock 
8.Easy programming with large instruction set
9.Large programming memory
