HC-A8C, is high performance modular PLC. It is the original dual system bus, and now become much faster. The extensible compact PLC is designed to seamlessly control communication network and analog positioning system. With maximum of 384 controllable local and networked Input / Output via CC-Link / AS-i, the HC-A8C uses its power and flexibility to provide solutions for variety of applications. 

1.3rd generation compact PLC 
2.Modular PLC
3.High efficiency with more speed, more performance, more memory, and new functions. 
4.Built-in high speed processing and positioning 
5.Exchangeable communication expansion boards that mount directly into the main unit (USB, RS-232C, RS-422, RS-485) 
6.Simultaneous memory cassette and display module mounting available 
7.Integrated real-time clock 
8.Easy programming with large instruction set and table positioning configuration 
9.Large programming memory
