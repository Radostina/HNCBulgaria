TP 2543 Series Human Machine Interface

The TP2000 Series Human Machine Interface (HMI) provides various touch screens with multiple dimensions and colors. It also offers fast and convenient control functions for industrial automation machines. In addition, our RTOS-based and user-friendly Soft Screen Editor and Programming Software configures the whole series. Users can quickly edit images and graphs and easily set suitable communication protocol. More applications can be created, edited, downloaded and uploaded.
