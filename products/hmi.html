HMI

HNC Electric provides various Human Machine Interface (HMI), including touch screen and text panel with different size and resolution. The new TP2000 series HMI can support all brands of PLCs with internal protocol. It also offers fast and convenient control functions for industrial automation machines. In addition, our RTOS-based and user-friendly Soft Screen Editor and Programming Software configures the whole series. Users can quickly edit images and graphs and easily set suitable communication protocol. More applications can be created, edited, downloaded and uploaded.

2.Resolution: 800*480 / 320*240 / 288*96 / 160*64

3.Communication ports: USB / RS 485 / 422 / 232

4.USB disk and Ethernet option
