HNC-818BM CNC controller for Milling Machine

This series of products is a full digital bus-type high - grade numerical control device with a modular and open architecture based on NCUC industrial fieldbus technology with independent intellectual property rights.Support bus all-digital servo motor, servo drive unit and absolute value type support bus remote I/O units, handheld unit interface integration, electronic disk storage program, support CF card, USB, Ethernet, etc. The function extension and data exchange.Adopt the 10.4 'LED display.It is mainly used in nc milling machine and milling center. 

1. True color graphics interface design, adapt 10.4 "color LED display, resolution of 800 * 800, fault diagnosis and alarm, graphical display and simulation of machining track, simple operation, easy to master and use.

2. Support multi-axis multi-channel, maximum 9 feed /channel.

3. A variety of digital AC servo drive unit and spindle motor (synchronous, asynchronous, linear, torque motor) can be selected.

4. Ladder diagram online monitoring and editing, the storage of block diagrams (the interface is arbitrary, graphics are not lost).

5. Support the handheld cell interface.

6. Support the function of the gantry shaft synchronization, dynamic shaft release/capture, and channel synchronization.

7. The interpolation cycle is 4ms ~ 0.5ms, and the minimum input unit is 10-4mm/deg/inch.

8. Inherited the HNC-21 series of powerful macro program, and has further expanded, users can use more variables, functions, and increase the modal user macro program calls and a series of advanced features.

9. Simplified programming functions: mirroring, zooming, rotating, direct drawing dimension programming, etc.

10. Processing breakpoint saving/recovery function, reverse clearance and single, two-way pitch error compensation function.

11. Built-in RS232 interface, easy to realize machine tool CNC communication, support high-speed Ethernet data exchange.

12. 1MB program power outage storage area can be extended with CF card, maximum to 2GB, support USB hot plug.

13. 512MB RAM processing memory buffer, custom G code function.

14.Background editing and blueprint programming (optional), using international standard G code programming, compatible with various popular CAD/CAM automatic programming systems.

15. With straight line interpolation, arc interpolation, polar coordinate interpolation, interpolation, cylinder helix interpolation, etc., to support rotation, scaling, mirroring, fixed cycle, thread cutting, cutting tool compensation, the user macro program, soft limit, etc.

16. Support the function of the gantry shaft synchronization, dynamic shaft release/capture, and channel synchronization.

17. Small line segment continuous processing function, especially suitable for CAD/CAM design complex mould parts processing.

18. Using the bus type PLC I/O unit, the maximum input/output support is 2048, and the maximum distance between the bus equipment can be up to 50 meters.

19. Fully functional, can realize various internal circulation.
